## 快速启动

### 环境要求
- Node.js 18+
- npm 9+
- 现代浏览器（Chrome / Edge / Firefox）
- 推荐屏幕宽度：1024px 以上

### 安装步骤

```bash
# 1. 克隆项目（如果还没有）
git clone <仓库地址>
cd rl-odyssey

# 2. 安装依赖（仅首次需要）
npm install

# 3. 启动开发服务器
npm run dev
```

启动成功后，终端会显示类似：

```
VITE v8.x.x  ready in 250 ms

➜  Local:   http://localhost:5173/
```

> **注意**：如果 5173 端口已被占用，Vite 会自动切换到 5174 或其他可用端口。

在浏览器中打开提示的本地地址即可开始游戏。

## 游戏整体流程

```
首页（启航）
    ↓
出发港教程（约5分钟，必须完成）
    ↓
世界地图（选择航线）
    ↓
价值群岛（当前可玩，推荐首次游玩）
    ├── 老虎机港 → Q表港 → SARSA港 → 深度港 → 双子港 → 决斗港
    └── BOSS：贪婪海盗（完成所有关卡后解锁）
    ↓
（未来）策略火山岛 / 连续冰川岛 / 融合之港
```

**核心循环**：每个关卡都包含以下8个步骤——
1. **故事（30秒）** — 像素风动画剧情，带入问题情境
2. **概念入门（30秒，可跳过）** — 学习本关所需的前置知识
3. **感受问题（2分钟）** — 亲手体验，感受算法要解决的难题
4. **认识算法（3分钟）** — 用通俗语言和可视化讲解算法原理
5. **观察学习（2分钟）** — 实时观看算法自主训练的过程
6. **挑战任务（5分钟）** — 完成本关核心挑战，获取悬赏奖励
7. **总结与奖励（1分钟）** — 获得金币、算法卡牌，回顾核心概念
8. **启航！** — 解锁下一关卡，出发

## 出发港教程

**URL**：`/tutorial`
**预计时间**：约 5 分钟
**是否可跳过**：否（必须完成才能进入地图）

### 教程步骤

#### Step 0 — 选择像素宠物
在进入教程之前，选择你的学习伙伴：
- 🐕 **像素狗**
- 🐱 **像素猫**
- 🤖 **像素机器人**

点击你喜欢的宠物，然后继续。

#### Step 1 — 亲自控制（1分钟）
在一个 4×4 的网格地图上，你直接操控宠物寻找零食。
- 使用**方向键**或**点击格子**来移动
- 找到零食，感受"手动寻路"的过程

#### Step 2 — 蒙眼行动（1分钟）
屏幕变暗，你只能凭记忆或猜测行动。
- 引出核心问题：**"如果能记住什么方向通常更好，会怎样？"**

#### Step 3 — 学习循环（2分钟）
观看宠物通过以下循环**自动学习**：

| 步骤 | 游戏中的叫法 | 强化学习术语 |
|------|------------|------------|
| 看看周围 | 你在哪里 | 状态（State） |
| 选择方向 | 做出决定 | 动作（Action） |
| 获得反馈 | 找到零食😊/碰墙😤/空格😐 | 奖励（Reward） |
| 记住结果 | 下次记得 | 学习（Learning） |

一张可视化的"记忆热图"会随着学习进展自动填充。

#### Step 4 — 选择航线（1分钟）
教程完成！世界地图解锁，系统自动解锁**老虎机港**（价值群岛第一站）。

> **提示**：教程每个步骤都是存档点。如果中途关闭页面，再次打开会从最后完成的步骤继续。

### 老虎机港（多臂老虎机）

**URL**：`/port/bandit`
**解锁条件**：完成教程
**算法**：Multi-Armed Bandit（多臂老虎机）

**故事背景**
你来到一个神秘的海湾，岸边排列着五个宝箱，每个宝箱发出不同颜色的光。一位老海盗说："五个宝箱，一百次机会。你能找出哪个给的最多吗？"

**玩法步骤**

1. **感受问题**：点击宝箱，每次获得随机金币奖励。100次机会内，靠直觉决定点哪个。
2. **认识算法**：
   - **探索（Exploration）**：尝试不熟悉的宝箱，收集更多信息
   - **利用（Exploitation）**：坚持点已知最好的宝箱
   - **ε-贪婪策略**：大部分时间选最优（利用），偶尔随机尝试（探索）
   - **UCB策略**：给尝试次数少的宝箱额外加分，主动保持好奇
3. **观察学习**：观看算法自动运行，看奖励曲线如何上升
4. **挑战任务：赌徒的困境**
   - 目标：100次机会内，比纯随机策略赚更多金币
   - **注意**：宝箱在第50轮会悄悄改变奖励规律（非稳态老虎机）
   - 通关奖励：100~500 金币

**悬赏等级目标**

| 等级 | 目标 | 金币倍率 |
|------|------|--------|
| C 新手 | 比随机高 1.1 倍 | 0.5× |
| B 标准 | 比随机高 1.5 倍 | 1× |
| A 高级 | 比随机高 2.0 倍 | 2× |
| S 传奇 | 比随机高 2.5 倍+ | 3× |

**解锁奖励**：卡牌「探险家的直觉」

### Q表港（Q学习）

**URL**：`/port/qtable`
**解锁条件**：完成老虎机港
**算法**：Q-Learning（Q学习）

**故事背景**
你来到一座被迷雾笼罩的诅咒之岛。老水手警告：这座岛变幻莫测，只有能绘制每条路径的制图师才能逃脱。

**玩法步骤**

1. **感受问题**：在网格地图上手动控制角色，尝试找到出口（避开陷阱）。
2. **认识算法**：Q表格的工作原理
   - Q表格 = 一张"每个位置每个方向的得分表"
   - **贝尔曼方程**（通俗版）：当前分数 = 眼前奖励 + 折扣 × 下一步最好结果
   - 比喻：**向未来的自己征求意见**
3. **观察学习**：实时看到Q值热图随着训练变化——好的路线越来越亮
4. **挑战任务：逃离诅咒之岛**
   - 你先手动走20步，设定基准
   - 然后训练Q学习——它能找到比你更短的路吗？
   - 通关奖励：200~800 金币

**悬赏等级目标**

| 等级 | 目标 |
|------|------|
| C | Q学习路径长度 ≤ 你的手动路径 |
| B | 比你短 20% |
| A | 比你短 40% |
| S | 找到最优路径 |

**解锁奖励**：卡牌「制图师」

### SARSA港（SARSA）

**URL**：`/port/sarsa`
**解锁条件**：完成Q表港
**算法**：SARSA（同策略时序差分）

**故事背景**
你来到一条山路，捷径就在悬崖边。一位老探险家警告：快路很诱人，但很危险。聪明的领航员从每一次失误中学习——而不只是理论上的最佳行动。

**玩法步骤**

1. **感受问题**：在靠近悬崖的网格上行走，体验"走捷径"的诱惑与风险
2. **认识算法**：SARSA 与 Q-Learning 的核心区别
   - **Q-Learning（离策略）**：学的是"如果我能完美行动，我应该怎么做"
   - **SARSA（同策略）**：学的是"我实际上会怎么做，并从真实体验中学习"
   - 比喻：Q-Learning 是看别人的日记学习，SARSA 是写自己的日记学习
3. **观察学习**：左右对比 SARSA 和 Q-Learning 在悬崖地图上的行为差异
4. **挑战任务：悬崖赛跑**
   - 同样的悬崖地图，SARSA 和 Q-Learning 同时出发
   - 目标：SARSA 安全到达终点，而 Q-Learning 因走捷径而坠崖
   - 通关奖励：200~600 金币

**解锁奖励**：卡牌「谨慎者」

### 深度港（DQN）

**URL**：`/port/deep`
**解锁条件**：完成SARSA港
**算法**：Deep Q-Network（深度Q网络）

**故事背景**
你已经绘制了宝岛的地图，但现在面对着一片广阔的可能性之海。一位老领航员说："当地图大到画不下时，你需要一个能看到规律的大脑。欢迎来到神经网络的世界。"

**玩法步骤**

1. **概念入门（重要）**：什么是神经网络？
   - 比喻：一台智能图案匹配机器
   - 它接收像素画面，输出"每个动作的预期得分"
2. **感受问题**：亲自玩像素版 Breakout（打砖块），感受"状态空间太大"的挑战
3. **认识算法**：DQN 的三个关键创新
   - **神经网络代替Q表格**：处理高维输入（图像）
   - **经验回放**：比喻为"打乱复习笔记"——随机采样历史经验防止遗忘
   - **目标网络**：比喻为"稳定的老师"——延迟更新避免训练不稳定
4. **观察学习**：观看DQN训练，神经网络的价值预测叠加在游戏画面上
5. **挑战任务：像素街机挑战**
   - 你先手动玩 Breakout，创下高分基准
   - 调整"经验池大小"和"学习率"，训练DQN智能体
   - 目标：DQN得分超过你的手动高分
   - 通关奖励：500~1500 金币 + 船体部件「强化船壳」

**悬赏等级目标**

| 等级 | 目标 |
|------|------|
| C | DQN 得分 ≥ 玩家手动得分 |
| B | DQN 得分 ≥ 1.2× 玩家 |
| A | DQN 得分 ≥ 1.5× 玩家 |
| S | DQN 得分 ≥ 2× 玩家 |

**解锁奖励**：卡牌「神经领航员」

### 双子港（双重DQN）

**URL**：`/port/double-dqn`
**解锁条件**：完成深度港
**算法**：Double DQN（双重深度Q网络）

**故事背景**
你发现了一个古老的神谕，声称知道每个宝藏的价值。但传言说神谕总是给出最高估计——从不诚实。为了找到真相，你需要第二个意见。双重DQN：检查自身乐观情绪的算法，登场了。

**玩法步骤**

1. **感受问题**：看到 DQN 的Q值被大幅高估——算法以为某条路很好，实际上并不是
2. **认识算法**：为什么 DQN 会过度乐观？
   - **问题**：DQN 用同一个网络选动作和评估价值，产生系统性高估偏差
   - **解决**：用两个网络相互制衡——一个选动作，另一个评估价值
   - 比喻："不要只听一个人的意见"
3. **观察学习**：柱状图对比展示——DQN vs Double DQN 的Q值估计误差
4. **挑战任务：怀疑者的试炼**
   - 在演示中识别哪些Q值被高估了
   - 对比两种算法在相同环境下的长期表现
   - 通关奖励：300~700 金币

**解锁奖励**：卡牌「怀疑论者」

### 决斗港（决斗DQN）

**URL**：`/port/dueling-dqn`
**解锁条件**：完成双子港
**算法**：Dueling DQN（决斗深度Q网络）

**故事背景**
你已经学会了给动作打分。但有一个更深的问题：一个动作的好坏是因为你在哪里，还是你做了什么？一位睿智的分析师把两者分开——分别理解状态的内在价值和每个动作的优势。决斗DQN，登场了。

**玩法步骤**

1. **感受问题**：在某些情况下，不论做什么动作结果都一样（状态本身决定好坏）
2. **认识算法**：决斗DQN 的网络架构分解
   - **状态价值 V(s)**：这个位置本身有多好？（与动作无关）
   - **优势函数 A(s,a)**：在这个位置，这个动作比平均水平好多少？
   - Q(s,a) = V(s) + A(s,a)
   - 比喻：把"你在哪里"和"你做了什么"的贡献分别算清楚
3. **观察学习**：可视化展示网络内部的 V 流和 A 流如何分别激活
4. **挑战任务：分析师的考验**
   - 在视觉谜题中，分辨状态价值和动作优势的不同贡献
   - 对比 DQN 和 Dueling DQN 的学习效率差异
   - 通关奖励：300~700 金币

**解锁奖励**：卡牌「分析师」

### BOSS：贪婪海盗

**URL**：`/boss/greedy-pirate`
**解锁条件**：完成价值群岛全部6个关卡
**类型**：迷你BOSS战

**故事背景**
一个海盗总是选择眼前最高奖励——纯粹贪心，从不考虑长远。你训练的智能体能打败他吗？

**玩法**
- 设计一个**延迟奖励迷宫**：眼前有一个小奖励，绕路才能得到大奖励
- 纯贪心策略：直接去小奖励，快但总分低
- 你训练的 Q-Learning / DQN 智能体：能学会先绕路，最终得到更高总分

**通关奖励**：1000 金币 + 卡牌「耐心的策略师」

## 关卡解锁关系

```
教程完成
    ↓
老虎机港（bandit）
    ↓ 完成后解锁
Q表港（qtable）
    ↓ 完成后解锁
SARSA港（sarsa）
    ↓ 完成后解锁
深度港（deep）
    ↓ 完成后解锁
双子港（double-dqn）
    ↓ 完成后解锁
决斗港（dueling-dqn）
    ↓ 全部完成后解锁
BOSS：贪婪海盗（boss/greedy-pirate）
```

**重要规则**：
- 每个关卡需要完成**上一个关卡的全部8步**（包括"启航"步骤）才会解锁
- 未解锁的关卡在岛屿页面显示为灰色锁定状态
- 所有关卡可以**无限次重玩**，不消耗任何资源
- 每次重玩都可以刷新悬赏等级，提升到更高的 A/S 评级

### 关卡技巧

**通用技巧**：
- 使用「跳到挑战」按钮可以在第2次游玩时直接跳到挑战任务，节省时间
- 调整**学习率（Learning Rate）**时，从中间值开始（如 0.1），太高会不稳定，太低学习太慢
- 调整 **ε（探索率）**时，高 ε = 多探索，低 ε = 多利用；训练前期需要高探索

**老虎机港技巧**：
- 前50轮广泛探索所有宝箱
- 第50轮后宝箱奖励会改变，需要重新探索适应变化
- UCB 策略比 ε-贪婪更能适应这种变化

**Q表港技巧**：
- 观察热图：橙色/红色 = Q值高，蓝色 = Q值低
- 折扣因子（γ）调高会让智能体更有远见；调低只看短期

**深度港技巧**：
- 经验池（Replay Buffer）太小 → 过拟合近期经验；建议至少1000
- 学习率太高（> 0.01）→ 训练震荡，降低到 0.001 更稳定

### 性能提示

- 如果动画卡顿，尝试降低训练速度（将速度滑块调到 1x 或 2x）
- Canvas 渲染密集时，关闭其他浏览器标签可以改善流畅度
- 浏览器控制台中的无害提示（React DevTools、i18next）不影响游戏

## 存档与进度

### 自动存档
游戏进度（解锁关卡、金币、卡牌、段位）会**自动保存**到浏览器的 localStorage。
关闭页面、刷新浏览器均不会丢失进度。

### 继续游玩
重新打开游戏时：
- 若教程已完成 → 直接显示「继续冒险」按钮，跳过首页
- 若有未完成的关卡 → 从当前关卡的**最后保存步骤**继续
- 每个关卡步骤都是一个存档点

### 清除存档
如果需要重新开始（清除所有进度）：
1. 打开浏览器开发者工具（F12）
2. 进入 Application → Local Storage → `http://localhost:5173`
3. 删除 `rl-odyssey-game` 和 `rl-odyssey-cards` 两个键
